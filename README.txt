FINOVANTA — Soul-inspired website v4

This is a clean static website inspired by the visual direction of soul.io / Landing Love: black background, minimal monochrome typography, technical geometry, dotted frames and small protocol-style labels.

Files:
- index.html
- privacy.html
- terms.html
- styles.css
- script.js
- assets/favicon.svg

How to deploy on Netlify:
1. Unzip the archive.
2. Drag the finovanta_soul_like_v4 folder into Netlify Deploys.
3. Connect your custom domain.

How to edit:
- Main text: index.html
- Design: styles.css
- Effects/runtime/form message: script.js
